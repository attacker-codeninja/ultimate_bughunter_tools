# Installation
- git clone https://github.com/dreamer1eh/ultimate_bughunter_tools
- cd ultimate_bughunter_tools
- chmod +x install.sh
- ./install.sh
- cd ~/ultimate_bht       
           *(All tools are available in this directory)
